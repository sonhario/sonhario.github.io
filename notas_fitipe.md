## Discussões Técnicas

### Por que Supabase vs Google Drive?

**Supabase escolhido porque:**
- Não é só storage - é banco de dados + storage + API REST
- PostgreSQL para queries complexas (busca, moderação, tags)
- API automática gerada das tabelas
- 1GB storage gratuito + 500MB database
- URLs públicas diretas para embed
- Google Drive: não tem database, URLs complicadas, limites bandwidth

**Decisão:** Supabase Free Tier
